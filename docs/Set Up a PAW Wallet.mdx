PAW Wallet allows users to manage digital assets across multiple networks with a simple, secure interface. This guide will walk you through everything you need to know to set up your wallet safely and get started.

:::tip

*Your seed phrase is the only way to recover your wallet. If you lose it, your funds are permanently unrecoverable. Always store it securely offline.*

:::


## Minimum Technology Requirements {#min-requirements}

Before setting up PAW Wallet, ensure your device meets the following requirements for smooth performance and security.

- **Mobile**
  - Android 8+ (Recommended: Android 12+)
  - iOS 8+ (Recommended: iOS 16+)
  - Minimum 2GB RAM

- **Desktop (Browser Only)**
  - Chrome, Firefox, Brave, or Edge (latest two versions)
  - Windows 8.1+ or macOS 10.13+ recommended for stability
  - To bridge assets, you will typically connect using an external wallet (e.g., MetaMask)

- **Network**
  - Stable internet connection
  - *Avoid using public Wi-Fi when creating your wallet*

## Secure Your Environment {#secure-your-environment}

Before setting up PAW Wallet, ensure your device meets the following requirements for smooth performance and security.

- Turn off screen sharing or recording apps.
- Cover any external cameras or use a privacy hood.
