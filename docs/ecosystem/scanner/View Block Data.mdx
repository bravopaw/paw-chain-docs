# View Block Data

## i. Accessing PAW Scanner

Open your preferred browser and visit: `https://scanner.pawchain.net`

## ii. Accessing Blocks

In the upper right-hand corner of the screen, click the **Blocks** button.

![](/img/scanner4.png)

This will open a list of **Layer 3** blocks shown in descending order based on creation time.

## iii. Searching for a Block

Use the **search bar** to find a specific block by its `block number`.

## iv. Viewing Block Details

Clicking on a specific block opens a detailed block page, which includes:

- `Block number`
- `Fee recipient address`
- `Status`
- `Timestamp`
- `Transferred token address`
