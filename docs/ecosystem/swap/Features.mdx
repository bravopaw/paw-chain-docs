# PAW Swap Features

## Swap Networks

You can choose the network you wish to swap to or from by pressing the dropdown icon in the trading box.

## Max Slippage

## Transaction Deadline

Define a time limit for how long the swap can remain pending before it's automatically cancelled. See [Setting Transaction Deadlines](/ecosystem/swap/slippage-and-deadline) for more.

## Send to PAW Wallet

When selecting **PAW Chain** as your receiving network, enter the **PAW Wallet** address you want the asset delivered to.

## Add Liquidity

Directs you to **PAW Wallet** to provide liquidity to a pool. See [Developer Mode](/ecosystem/wallet/developer-mode) for steps.
