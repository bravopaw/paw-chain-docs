# What is Developer Mode?

**Developer Mode** is a feature in **PAW Wallet** designed for token developers looking to onboard their projects to **PAW Chain**. Once enabled, Development Mode allows a tokenâ€™s full supply to be bridged into the PAW ecosystem.

It also provides direct access to the **Add Liquidity** function, enabling developers and holders to pair their token with **PAW Coin** to create or maintain a liquidity pool.

To get started:

- **Read:** [How to Launch a Token](/ecosystem/launchpad/launch-token)

# How to Enable or Disable Developer Mode

## i. Accessing PAW Wallet

Open your in-app wallet browser and go to: `https://wallet.pawchain.net`

## ii. Opening the Settings Menu

- Tap your **PAW Wallet** address in the top-right corner
- Select **Settings** from the dropdown menu

![](/img/dev1.png)

## iii. Enable or Disable Developer Mode

- Scroll to the bottom of the **Settings** page
- Press **Developer Mode**

![](/img/dev2.png)

A pop-up will appear asking whether to enable or disable the feature.
Press **Yes** to proceed, or **No** to cancel the action.

![](/img/dev3.png)
